.rotate { animation: rotate-animation 2s infinite; }
@keyframes rotate-animation {
0% { transform: rotate(0deg); }
100% { transform: rotate(360deg); }
}
Luego, para aplicar esta clase a un elemento HTML, puede usar el atributo class en el elemento, por ejemplo:
class="rotate"

Además de translate y rotate, los valores que se pueden usar en las reglas @keyframes incluyen
scale(),
skew(),
opacity,
color,
background-color,
font-size,
border-radius,
y muchos más.

Para alterar el punto de rotación usando @keyframes, puede especificar el valor de la propiedad transform-origin. Esta propiedad especifica el punto desde el cual se aplicará la transformación. Por ejemplo, si desea que el punto de rotación se centre en un elemento, puede usar el siguiente código:
@keyframes rotate { from {transform-origin: center;} to {transform: rotate(360deg);} }
También puede especificar coordenadas x,y para alterar el punto de rotación, por ejemplo:
@keyframes rotate { from {transform-origin: 50% 50%;} to {transform: rotate(360deg);} }
Esto alterará el punto de rotación al centro del elemento.
